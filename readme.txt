How to use (on eclipse) :

1. please unzip it first
2. import the project into eclipse IDE
3. main class is TaxResult.java
4. junit test is on "test" folder, with file name "ParameterTest.java"
5. run as java application for see the application result
6. run as junit test for see the test result
